Project 2

EduFord University Website

Inspired by a YouTube tutorial, the EduFord University Website project was a hands-on endeavor to master the intricacies of responsive design while refining HTML and CSS skills. The primary objective was to create a fully responsive multi-paged college website with a clean and organized codebase, maintaining a consistent styling theme for a professional aesthetic. Although predominantly HTML and CSS, the project introduced some elements of JavaScript and PHP.

Key Features

  • Strategic use of HTML for seamless page linkage.
  • Implementation of CSS rules for an engaging and cohesive visual interface, including background images and the application of pseudo-classes and elements.
  • Responsive design to ensure optimal performance on various screen sizes.
  • Introduction to JavaScript for handling menus on smaller screens.
  • Application of PHP for form handling on the website.
  • Emphasis on clean and organized code for a polished design.

Note: JavaScript and PHP, though applied, are still in the early stages of exploration as the primary focus remains on perfecting HTML and CSS skills, particularly in responsive design.

Technologies Used

Visual Studio served as the primary development environment for writing HTML, CSS, JavaScript, and PHP files.

Project 4

Personal Portfolio Website

Derived from a YouTube tutorial, the Personal Portfolio Website project was a significant step in mastering responsive design while refining HTML and CSS skills. The primary goal was to create a fully responsive single-paged portfolio website with a focus on mobile-first design principles. The inclusion of JavaScript, though minimal, added interactive elements to enhance user experience.

In an effort to elevate the portfolio, the website evolved from a single-paged structure to a portal showcasing all Code Canvas Web Development projects. Two additional pages, the Listings page and the Portfolio Portal Page, were integrated to provide a comprehensive overview. The project served as a platform to challenge and further develop skills, resulting in a website that aligns with the creator's satisfaction.

Key Features

  • Strategic use of HTML for seamless page linkage.
  • Implementation of CSS rules for an engaging and cohesive visual interface, incorporating images as backgrounds and applying pseudo-classes and elements.
  • Responsive design, adopting a mobile-first approach for a seamless experience on various devices.
  • Inclusion of JavaScript for menu handling on smaller screens and form interaction.
  • Development of a responsive timeline as the table of contents, enhancing navigation.

Technologies Used

Visual Studio was the chosen development environment for crafting HTML, CSS, and JavaScript files.
